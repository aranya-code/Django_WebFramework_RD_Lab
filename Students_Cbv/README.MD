## рҹ§  Model

```python
class studentInfo(models.Model):
    id = models.IntegerField(primary_key=True)
    name = models.CharField(max_length=100)
    score = models.FloatField()

    def get_absolute_url(self):
        return reverse('detail', kwargs={'pk': self.pk})
```

рҹ“Ң Each student has a unique ID, name, and score.

## рҹ”§ Views

Implemented using Django **Class-Based Views (CBVs)**:

- `studentInfoView` вҶ’ Displays all students  
- `studentDetailsView` вҶ’ Shows student details  
- `studentCreateView` вҶ’ Creates a new student  
- `studentUpdateView` вҶ’ Updates student data  
- `studentDeleteView` вҶ’ Deletes a student
